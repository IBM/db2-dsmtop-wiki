title: Throughput - Workloads

<h2>Purpose</h2>
<p>Show how much work is happening in each WLM workload.</p>
<p>One of the intended purposes of workloads in WLM is to split particular users or applications into separate categories for monitoring and reporting purposes.  For example, you might create a workload where all Cognos reports will run so that you can track the impact of Cognos on your production database.</p>
